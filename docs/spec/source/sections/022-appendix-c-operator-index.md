## Appendix C. Operator Index

Important operators:

| Group | Operators |
|--|--|
| Arithmetic | `+`, `-`, `*`, `/`, `%` |
| Bitwise | `&`, `|`, `^`, `~`, `<<`, `>>` |
| Comparison | `==`, `!=`, `===`, `!==`, `<`, `<=`, `>`, `>=` |
| Logical | `&&`, `||`, `!` |
| Assignment | `=`, `+=`, `-=`, `*=`, `/=`, `%=`, `&=`, `|=`, `^=`, `<<=`, `>>=` |
| Other | `..`, `??`, `?.`, `!`, `->` |

The arrow token is `->`.
