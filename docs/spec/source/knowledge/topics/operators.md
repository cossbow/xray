## Operators

### Arithmetic: `+`, `-`, `*`, `/`, `%`, `**` (power)
### Comparison: `==`, `!=`, `<`, `>`, `<=`, `>=`
### Logical: `&&`, `||`, `!`
### Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`
### Assignment: `=`, `+=`, `-=`, `*=`, `/=`, `%=`
### Increment/Decrement: `i++`, `i--` (variables only, not fields)
### Null coalescing: `x ?? default`
### Optional chaining: `obj?.field`
### Spread: `...arr`
### Type: `typeof(x)`, `x is Type`
